Get Started

StateFlowNG is set of open source flow navigation Angular UI components.

Download

StateFlowNG is available at npm, if you have an existing application run the following command to download it to your project.

npm install state-flow --save
Import

UI components are configured as modules, once StateFlowNG is downloaded and configured, modules and apis can be imported from primeng/{module} shorthand in your application code. Documentation of each component states the import path.

import {StateFlowModule} from 'state-flow/engine'; //accordion and accordion tab import {MenuItem} from 'state-flow/api'; //api

That is all, you may now import StateFlowNG components, for a working example visit the StateFlowNG CLI QuickStart sample at GitHub.
